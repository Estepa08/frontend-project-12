## 📦 Требования

- Node.js версии 18 или выше
- npm

## ⚙️ Установка и запуск

```bash
git clone https://github.com/your-username/frontend-project-12.git
cd frontend-project-12

make install
make build
make start
```

После установки зависимостей проект доступен по адресу:
**http://localhost:5001**
